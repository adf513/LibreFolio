# F-080 — Candlestick Chart / Volume Bars

## What it does
Adds a candlestick (OHLCV) chart view to the asset detail page, alongside the existing line chart. Volume bars shown below. OHLCV data already exists in `PriceHistory` (open, high, low, close, volume columns).

## Implementation notes
- OHLCV data already in DB: `PriceHistory.open`, `.high`, `.low`, `.close`, `.volume`
- Two new components needed: `CandlestickChart.svelte` and `VolumeBar.svelte`
- Stubs were created and then removed in Phase 6 Step 6 — needs clean reimplementation
- ECharts has native candlestick support (`series.type: 'candlestick'`)
- Medium priority
