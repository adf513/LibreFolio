# F-063 — dev.py CLI

## What it does
The single entry point for all development operations in LibreFolio. A Python CLI using `argparse` with `argcomplete` for shell autocompletion. Covers server management, database operations, frontend build, API client sync, testing, linting, i18n management, MkDocs documentation, and Docker operations.

## Layer breakdown
**Infra**: `dev.py` (root-level, ~1000+ lines), `scripts/` (imported submodule files: `cli_base.py`, `cli_tree_parser.py`, and command implementations)

## Implementation notes
- Autocompletion: `eval "$(register-python-argcomplete dev.py)"` in shell RC
- Key command groups: `server`, `db`, `front`, `api`, `test`, `i18n`, `mkdocs`, `docker`, `lint`, `format`
- `./dev.py server` — starts uvicorn dev server
- `./dev.py db create-clean` — drop + recreate DB from `001_initial.py` (no incremental migrations)
- `./dev.py api sync` — regenerates TypeScript Zodios client from OpenAPI schema
- `./dev.py i18n audit|add|remove|update|search` — i18n key management
- `./dev.py docker build` — full build pipeline (front + mkdocs + docker build)
