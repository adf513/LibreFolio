# F-032 — Asset List View dual view

## What it does
The `/assets` page showing all configured financial assets in two interchangeable views: a card grid (`AssetCard.svelte`) and a data table (`AssetTable.svelte`). View toggle persisted in localStorage. Each card shows asset type badge, display name, currency, current price, delta % for multiple periods, and a mini price chart. The page includes "Add Asset", "Sync All", and bulk-delete actions.

**Frontend**: `frontend/src/routes/(app)/assets/+page.svelte`, `frontend/src/lib/components/assets/AssetCard.svelte` (287 lines), `frontend/src/lib/components/assets/AssetTable.svelte` (265 lines), `frontend/src/lib/components/assets/AssetSyncModal.svelte`
