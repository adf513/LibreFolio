# F-033 — Asset Detail Page chart/signals/editor

## What it does
Per-asset detail page at `/assets/[id]` with three main sections: the full interactive price chart with signal overlay, the data editor for inline price/event editing, and the provider management panel. Users can sync, view OHLCV history, add/edit/delete events, configure providers, and run metadata refresh — all from this page.

## Layer breakdown
**Frontend**: `frontend/src/routes/(app)/assets/[id]/+page.svelte`, `frontend/src/lib/components/assets/AssetDataEditorSection.svelte` (493 lines), `frontend/src/lib/components/assets/ProviderAssignmentSection.svelte` (724 lines), `frontend/src/lib/components/assets/AssetPriceSummary.svelte`, `frontend/src/lib/components/assets/AssetModal.svelte` (edit mode), `frontend/src/lib/components/charts/PriceChartFull.svelte`
