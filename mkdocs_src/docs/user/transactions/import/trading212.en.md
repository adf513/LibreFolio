    This plugin is in **Beta** — tested with sample files but edge cases may exist.

## How to Export

1. Log in to [Trading212](https://www.trading212.com).
2. Go to **History → Export** and download the CSV.

## Notes

- Stocks, ETFs, and PIE transactions supported.
