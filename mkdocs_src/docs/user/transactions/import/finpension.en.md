## How to Export

1. Log in to [Finpension](https://www.finpension.ch).
2. Go to **Account → Transactions → Export** and download the CSV.

## Notes

- Swiss pension platform.
- Uses semicolon (`;`) as CSV delimiter — the plugin handles this automatically.
