# 📊 Tipi di Asset

LibreFolio supporta un'ampia gamma di classi di asset per coprire un portafoglio diversificato. Ogni tipo di asset presenta comportamenti specifici riguardanti la determinazione del prezzo, i dividendi e il trattamento fiscale.
