# F-066 — Zodios API Client

## What it does
Type-safe API client for the frontend generated from the backend's OpenAPI schema using Zodios. Provides full TypeScript type inference for all API requests and responses with Zod runtime validation. Running `./dev.py api sync` regenerates the client whenever the backend API changes.

## Layer breakdown
**Frontend**: `frontend/src/lib/api/` — generated Zodios client, OpenAPI-derived TypeScript types, Zod schemas

- (foundational frontend infrastructure — no feature dependencies)

## Enables
- (used by all frontend pages and components that call the API)

## Implementation notes
- Library: Zodios (OpenAPI → Zod → TypeScript types)
- Regeneration: `./dev.py api sync` — calls backend to export OpenAPI JSON, runs codegen
- Runtime validation: Zod schemas validate API responses at runtime — type errors surface immediately
- All API calls in the frontend go through this client — no raw `fetch()` calls
- `frontend/src/lib/api/` contains: generated client file + type definitions
