# 📱 Galerie Mobile

Découvrez l'interface mobile réactive de LibreFolio. Les captures d'écran s'adaptent automatiquement à votre thème et à votre langue sélectionnés.

!!! tip "Thème et Langue"

    Utilisez le **bouton de thème** dans l'en-tête (☀️/🌙) pour basculer entre les modes clair et sombre.
    Utilisez le **sélecteur de langue** (🇬🇧) dans l'en-tête pour afficher les captures d'écran dans différentes langues.

!!! info "Optimisé pour Mobile"

    LibreFolio est entièrement réactif et fonctionne parfaitement sur les smartphones et tablettes. L'interface s'adapte automatiquement aux écrans plus petits avec un menu de navigation repliable.
