# F-039 — RSI Signal

## What it does
Relative Strength Index momentum oscillator on a secondary Y-axis (0–100). Measures overbought/oversold conditions: RSI > 70 → overbought, RSI < 30 → oversold. Signal processing equivalent: duty-cycle/saturation indicator using Wilder's SMMA (α = 1/N). Displayed on a separate panel below the price chart.

## Implementation notes
- `signalType`: `rsi`, `category`: `indicator`, `icon`: 📊
- `yAxisIndex = 1` — secondary (independent scale 0–100, left axis)
- **Parameters**:
  - `period` (number, default 14, range 2–200, suffix `days`)
  - `overbought` (number, default 70, range 50–100)
  - `oversold` (number, default 30, range 0–50)
- Formula: `RSI = 100 − 100/(1 + RS)` where `RS = avgGain / avgLoss` (Wilder SMMA)
- Computed iteratively O(N)
- Docs: `financial-theory/technical-analysis/indicators/rsi.en.md`
