# F-038 — EMA Signal

## What it does
Exponential Moving Average overlay on price charts. Tracks the price trend by applying exponential smoothing (more weight to recent prices). Signal processing equivalent: first-order IIR low-pass filter with `α = 2/(N+1)`. Shorter period = faster reaction, longer period = more lag. Rendered on the primary Y-axis (same scale as price).

## Implementation notes
- `signalType`: `ema`, `category`: `indicator`, `icon`: 📉
- **Parameters**:
  - `period` (number, default 14, range 2–500, suffix `days`) — smoothing window
  - `offset` (number, default 0, range −100–100, suffix `%`) — vertical shift
- Formula: `y[n] = α·x[n] + (1−α)·y[n−1]` where `α = 2/(period+1)`
- Computed iteratively O(N) — one multiplication per data point
- `yAxisIndex = 0` (primary, same scale as price)
- Docs: `financial-theory/technical-analysis/indicators/ema.en.md`
