# F-023 — FX Chart & Signal Overlay

## What it does
The interactive line chart on the FX detail page showing historical rate data with overlaid technical analysis signals. Uses a single ECharts instance managed by `PriceChartFull.svelte`. Users can add/remove/configure up to 10 signal types, toggle between absolute and percentage mode, use the 2-click measure tool, and access chart settings (colors, line widths, date range).

## Layer breakdown
**Frontend**: `frontend/src/lib/components/charts/PriceChartFull.svelte` (967 lines, main chart), `frontend/src/lib/components/charts/ChartSignalsSection.svelte` (734 lines, signal management), `frontend/src/lib/components/charts/LineChart.svelte` (714 lines, ECharts core), `frontend/src/lib/components/charts/MeasurePanel.svelte`, `frontend/src/lib/stores/chartSettingsStore.svelte.ts` (2-level settings: global + per-pair)

## Dependencies
- [[F-017]] FX Rate Sync — data source
- [[F-038]] to [[F-045]] Signal Library — all overlay signal types

## Enables
- (visual analysis — no downstream functional dependencies)

## Implementation notes
- `chartSettingsStore` uses Svelte 5 runes (`$state`, `$derived`); session-lifetime only (not persisted)
- Two-level settings: global defaults + per-pair overrides
- Percentage mode: all series normalized to their own first value (0% = starting point)
- `ChartSignalsSection` renders 3 categorized dropdowns (indicator/comparison/benchmark) + style popovers
