# F-040 — MACD Signal

## What it does
Moving Average Convergence Divergence — a composite signal that renders three series on a dedicated tertiary Y-axis: the MACD line (fast EMA − slow EMA), the Signal line (EMA of MACD), and a histogram (MACD − Signal, green/red by sign). Detects momentum shifts and generates crossover buy/sell signals. Signal processing equivalent: band-pass filter isolating intermediate momentum frequency.

## Implementation notes
- `signalType`: `macd`, `category`: `indicator`, `icon`: 📶
- `yAxisIndex = 2` — dedicated MACD axis (auto-scaled by ECharts)
- **Parameters**:
  - `fastPeriod` (number, default 12, range 2–200, suffix `days`)
  - `slowPeriod` (number, default 26, range 2–500, suffix `days`)
  - `signalPeriod` (number, default 9, range 2–100, suffix `days`)
- Uses `renderMulti()` (composite signal — 3 RenderedSignals from one config card)
- Three EMA passes computed iteratively O(N)
- Histogram bar chart colored green (positive) / red (negative)
- Docs: `financial-theory/technical-analysis/indicators/macd.en.md`
