# F-043 — Asset Comparison Signal

## What it does
Overlay signal that plots another asset's price history on top of any chart. Complementary to [[F-042]] FxPairSignal — allows overlaying an asset price on either an FX chart or another asset chart. Percentage mode normalizes each series to its own starting value for meaningful cross-asset performance comparison.

## Layer breakdown
**Frontend**: `frontend/src/lib/charts/signals/AssetComparisonSignal.ts`

## Dependencies
- [[F-023]] FX Chart & Signal Overlay — chart host
- [[F-027]] Asset Data Sync — asset price data must be available

## Enables
- (cross-asset analysis — no downstream)

## Implementation notes
- `signalType`: `asset-comparison`, `category`: `comparison`, `icon`: 📊
- No `maxInstances` limit — multiple assets can be overlaid
- **Parameters**:
  - `assetId` (select, `dynamicOptionsKey: 'configuredAssets'`) — runtime-resolved dropdown listing all configured assets
- `params._resolvedData` and `params._assetDisplayName` injected by `PriceChartFull` before `computePoints()`; `_` prefix excludes from serialization
- Percentage normalization: each series normalized to its own first value (not base chart's p0)
