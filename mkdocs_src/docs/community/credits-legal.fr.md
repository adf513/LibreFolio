# ⚖️ Crédits et mentions légales

## 🏆 Crédits

LibreFolio est le fruit d'un travail réalisé avec passion, conçu pour renforcer la souveraineté des individus sur leurs données financières.

### 👥 Équipe centrale

* **Emanuele Alfano** - *Créateur et développeur principal*
 * Conception, architecture, backend (FastAPI), frontend (SvelteKit) et DevOps.
* **Nicoleta Alfano** - *Graphismes et design*
 * Conception du logo, palette de couleurs et identité visuelle.

### 🤖 Développement assisté par l'IA

Ce projet a été développé avec le soutien d'assistants de développement IA avancés, en utilisant une méthodologie structurée **Plan -> Dev -> Test -> Review**.

* **Méthodologie** : Chaque fonctionnalité débute par un plan écrit détaillé, suivi d'un développement itératif, de tests rigoureux et d'une revue.
* **Transparence** : Le dépôt contient l'historique complet de ces plans dans le dossier `LibreFolio_developer_journal`, servant de témoignage du processus de développement et de ressource pour d'autres développeurs.

### 🌐 Traductions de la documentation

La documentation multilingue (italien, français, espagnol) a été générée automatiquement à l'aide de [**Aphra**](https://github.com/DavidLMS/aphra) — un flux de travail de traduction agentique open-source basé sur des LLM, conçu par David LMS. Le pipeline de traduction intègre le processus Analyze → Translate → Critique → Refine d'Aphra avec un système de diff structurel personnalisé pour l'assurance qualité.
