# F-099 — Borsa Italiana Asset Provider

## What it does
Adds a new asset source provider for the Italian stock exchange (borsaitaliana.it). Users can search, track prices, and fetch historical OHLCV data for stocks, bonds, and ETFs listed on Borsa Italiana.

## Layer breakdown
**Backend**: New provider class `BorsaItalianaProvider` (541 lines) implementing all `AssetSourceProvider` ABC methods: `search()`, `get_current_value()`, `get_historical_data()`, `get_metadata()`, `shutdown()`. Uses `borsa-italiana-scraping` library via shared `Sessione` HTTP session.

**Frontend**: Provider appears in `AssetSearchAutocomplete` dropdown and `ProviderAssignmentSection` form (auto-discovered via registry pattern).

## Dependencies
- [[F-059]] Provider Registry Pattern — uses `@register_provider` decorator for auto-discovery
- [[F-025]] Asset Provider Registry — implements `AssetSourceProvider` ABC

- (no direct dependents)

## Implementation notes
- **Optional dependency**: `borsa-italiana-scraping` in Pipfile; if not installed, provider is silently unavailable (`BORSA_ITALIANA_AVAILABLE = False`)
- **Shared session**: single `Sessione` instance reused across all calls; properly closed on `shutdown()`
- **Graceful degradation**: all methods guard with `if not BORSA_ITALIANA_AVAILABLE: raise AssetSourceError(...)`
- **Asset types mapped**: maps library's instrument types to LibreFolio's `AssetType` enum (STOCK, BOND, ETF)
- **Largest single-file provider** in the codebase (541 lines) — Borsa Italiana API is more complex than yfinance/JustETF
