!!! info "Beta"

    This plugin is in **Beta** — tested with sample files but edge cases may exist.

## How to Export

1. Log in to [Degiro](https://www.degiro.eu).
2. Go to **Account → Reports** and export your **Account Statement** as CSV.

## Notes

- Multi-language reports supported (EN, NL, IT, etc.).
- Dividends and transactions are in separate sections — upload both if needed.
