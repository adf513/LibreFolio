# 🏦 Broker

Un broker in LibreFolio rappresenta un conto di intermediazione — il posto in cui sono custoditi i tuoi investimenti (es. Interactive Brokers, Degiro, un conto bancario).

Tutte le transazioni, i report e i dati importati sono associati a un broker. È necessario almeno un broker per iniziare a monitorare il tuo portafoglio.

## ➕ Creare un broker

1. Vai alla pagina **Broker** dalla barra laterale
2. Clicca **"Nuovo broker"**
3. Compila i dettagli: nome, valuta base e, opzionalmente, un'icona
 <div class="screenshot-container" style="max-width: 700px; margin: 1rem auto;">
 <img class="gallery-img" data-category="brokers" data-name="edit-modal" alt="Modale di creazione/modifica broker" style="width: 100%; border-radius: 8px; box-shadow: 0 4px 16px rgba(0,0,0,0.1);">
 </div>

4. Il broker apparirà nella tua lista, pronto a ricevere transazioni e report

## 🔧 Cosa puoi fare con un broker?

| Funzionalità | Descrizione | Stato |
|---------|-------------|--------|
| **Visualizza dettagli** | Nome, valuta, data di creazione, icona | ✅ Disponibile |
| **Carica report** | File CSV/Excel dal tuo broker reale | ✅ Disponibile |
| **Condividi accesso** | Invita altri utenti con permessi basati sui ruoli | ✅ Disponibile |
| **Importa transazioni** | Analisi automatica dei report caricati tramite BRIM | 🔜 Prossimamente |
| **Visualizza transazioni** | Elenco, filtra e gestisci le operazioni importate | 🔜 Prossimamente |

!!! note "Work in Progress"

    La pagina dei dettagli del broker e l'interfaccia di gestione delle transazioni sono in fase di sviluppo attivo. Attualmente puoi creare broker, caricare report e condividere accessi. L'importazione e la visualizzazione complete delle transazioni saranno disponibili nelle prossime fasi.
