Experimente la interfaz completa de escritorio de LibreFolio. Las capturas de pantalla se adaptan automáticamente al tema e idioma seleccionados.

!!! tip "Tema e Idioma"

    Utilice el **interruptor de tema** en el encabezado (☀️/🌙) para cambiar entre el modo claro y oscuro.
    Utilice el **selector de idioma** (🇬🇧) en el encabezado para ver las capturas de pantalla en diferentes idiomas.
