## Aperçu

Le Tableau de bord est la **page d'accueil principale** après la connexion. Il fournit une vue consolidée de l'ensemble de votre portefeuille tous courtiers confondus — valeur nette, allocation, performance récente et widgets d'accès rapide.

Les widgets prévus incluent :

- **Valeur Nette** — Valeur totale du portefeuille dans votre devise de base
- **Répartition de l'Allocation** — Graphiques d'allocation par type d'actif et de distribution sectorielle
- **Aperçu des performances** — Rendements sur des plages temporelles sélectionnables (1S, 1M, 3M, YTD, 1A, Tout)
- **Transactions Récentes** — Les 5 à 10 dernières transactions tous courtiers confondus
- **Mouvements du Marché** — Vos positions triées par variation quotidienne en %
- **Exposition FX** — Résumé de l'exposition aux devises

---

## Vérification de la Connectivité

LibreFolio vérifie automatiquement la **connectivité du backend** lorsque vous ouvrez le Tableau de bord. Si le backend est inaccessible, une bannière s'affiche avec un bouton de reconnexion — aucune donnée obsolète ne s'affiche.
