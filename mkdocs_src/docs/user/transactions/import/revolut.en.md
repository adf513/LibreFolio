!!! info "Beta"

    This plugin is in **Beta** — tested with sample files but edge cases may exist.

## How to Export

1. Log in to [Revolut](https://www.revolut.com).
2. Go to **Stocks → Statements** and export the transaction history as CSV.

## Notes

- Revolut Trading accounts only (not the main card account).
- Multi-currency amounts in the same file are handled automatically.
