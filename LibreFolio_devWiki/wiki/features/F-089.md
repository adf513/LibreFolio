# F-089 — FX Provider Per-Plugin Documentation

## What it does
Dedicated MkDocs pages for each FX provider plugin: ECB, FED, BOE, SNB. Each page covers data coverage (currencies, date range), sync frequency, API source URL, and rate format. The `docs_url` property already exists on each provider class.

## Implementation notes
- One page per provider: `mkdocs_src/docs/developer/backend/fx/ecb.en.md` etc.
- `docs_url` property in each provider class already points to intended target
- Low priority
