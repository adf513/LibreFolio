# F-083 — Multi-File Multi-Broker Import

## What it does
Enables importing multiple files from multiple brokers in a single session. Files are parsed in parallel across different broker plugins, results are checked and validated per-broker, and the user can navigate between broker result sets before committing.

## Implementation notes
- Parallel file parsing: run BRIM plugins concurrently (thread pool)
- Per-broker validation step before commit
- UI navigation between broker result tabs
- HIGH priority — Phase 7 core feature
