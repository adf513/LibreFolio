# F-088 — Return-over-N Chart

## What it does
For each data point in the price history, show the gain/loss percentage if the asset had been bought N days prior. This creates a "rolling N-day return" chart. The user can select N (e.g. 30, 90, 365 days).

## Implementation notes
- Computed client-side from existing price history data
- N selector (preset buttons + custom input)
- Medium priority
