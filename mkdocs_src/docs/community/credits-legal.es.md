# ⚖️ Créditos y Aspectos Legales

## 🏆 Créditos

LibreFolio es un trabajo realizado por pasión, creado para empoderar a las personas con la soberanía de sus datos financieros.

### 👥 Equipo Principal

* **Emanuele Alfano** - *Creador y Desarrollador Principal*
 * Concepto, Arquitectura, Backend (FastAPI), Frontend (SvelteKit) y DevOps.
* **Nicoleta Alfano** - *Arte y Diseño*
 * Diseño del logotipo, paleta de colores e identidad visual.

### 🤖 Desarrollo Asistido por IA

Este proyecto fue desarrollado con el apoyo de asistentes de codificación de IA avanzada, utilizando una metodología estructurada de **Plan -> Desarrollo -> Pruebas -> Revisión**.

* **Metodología**: Cada funcionalidad comienza con un plan escrito detallado, seguido de un desarrollo iterativo, pruebas rigurosas y revisión.
* **Transparencia**: El repositorio contiene el historial completo de estos planes en la carpeta `LibreFolio_developer_journal`, sirviendo como testimonio del proceso de desarrollo y como recurso para otros desarrolladores.

### 🌐 Traducciones de la Documentación

La documentación multilingüe (italiano, francés, español) fue generada automáticamente utilizando [**Aphra**](https://github.com/DavidLMS/aphra), un flujo de trabajo de traducción basado en agentes y LLM de código abierto creado por David LMS. El pipeline de traducción integra el flujo de trabajo Analyze → Translate → Critique → Refine de Aphra con un sistema de diff estructural personalizado para el aseguramiento de la calidad.
