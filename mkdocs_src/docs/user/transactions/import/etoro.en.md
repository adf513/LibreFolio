!!! info "Beta"

    This plugin is in **Beta** — tested with sample files but edge cases may exist.

## How to Export

1. Log in to [eToro](https://www.etoro.com).
2. Go to **Portfolio → History → Account Statement** and export as CSV.

## Notes

- Supports stock and CFD transactions.
- Crypto trades may have limited detail.
