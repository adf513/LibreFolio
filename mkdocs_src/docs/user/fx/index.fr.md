# 💱 Taux de change (FX)

LibreFolio inclut un système complet de **gestion des devises (Forex / FX)**. Il vous permet de suivre les taux de change entre n'importe quelle paire de devises, avec une synchronisation automatique depuis les sources officielles des banques centrales.

---

## 📋 Page Liste des Taux

Accédez à **Taux de change** depuis la barre latérale pour voir toutes vos paires de devises configurées :

<div class="screenshot-container" style="max-width: 700px; margin: 1rem auto;">
 <img class="gallery-img" data-category="fx" data-name="list" alt="Page Liste des Taux" style="width: 100%; border-radius: 8px; box-shadow: 0 4px 16px rgba(0,0,0,0.1);">
</div>

Chaque carte affiche :

- 🏷️ La **paire de devises** avec des drapeaux (par ex. 🇪🇺 EUR → 🇺🇸 USD)
- 📈 Le **dernier taux** et la tendance
- 🏛️ Le **prestataire de données** (BCE, FED, BOE, etc.)
- 📊 Une **mini-courbe sparkline**

Vous pouvez **filtrer** par devise pour trouver rapidement une paire spécifique :
