# F-035 — CSS Scraper Provider

## What it does
An asset data provider that fetches current prices by scraping a web page using a user-supplied CSS selector. Supports both US (1,234.56) and EU (1.234,56) number formats. Used for assets not covered by yfinance or JustETF — for example, a bank's fund page, a real estate crowdfunding platform, or any custom price source accessible via HTTP.

The provider requires three `provider_params`: `current_css_selector` (CSS selector targeting the price element), `currency` (ISO 4217 code), and optionally `decimal_format` (US or EU).

## Layer breakdown
**Backend**: `backend/app/services/asset_source_providers/css_scraper.py`, optional dependencies: `httpx` + `BeautifulSoup`

## Implementation notes
- `provider_code`: `css_scraper`
- `accepted_identifier_types`: `[ProviderInputType.URL]` — the "identifier" is the full page URL
- Does NOT support history or asset events (`get_history_value()` raises `AssetSourceError`)
- `SCRAPER_AVAILABLE` flag: if `httpx`/`beautifulsoup4` not installed, provider degrades gracefully
- `params_schema` drives the dynamic form in `ProviderAssignmentSection.svelte`
