# F-084 — Transaction Gain Chart

## What it does
A chart showing per-buy-lot gain/loss over time with a dual Y-axis. For each open position, the gain is tracked from purchase date. Includes a FIFO/LIFO/PMC selector to switch the cost-basis method for visualization.

## Implementation notes
- Dual Y-axis: left=price, right=gain%
- Per-buy lot gain tracking requires FIFO/LIFO/PMC calculation per lot
- Phase 8 target — requires transaction model (Phase 7) to be complete
