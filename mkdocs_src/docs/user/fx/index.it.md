# 💱 Tassi di Cambio (FX)

LibreFolio include un sistema completo di gestione **Foreign Exchange (FX)**. Ti consente di tenere traccia dei tassi di cambio tra qualsiasi coppia di valute, con sincronizzazione automatica da fonti ufficiali delle banche centrali.

---

## 📋 La Pagina dell'Elenco FX

Vai su **Tassi di Cambio (FX)** dalla barra laterale per visualizzare tutte le coppie di valute configurate:

Ogni scheda mostra:

- 🏷️ La **coppia di valute** con emoji di bandiere (es. 🇪🇺 EUR → 🇺🇸 USD)
- 📈 L'**ultimo tasso** e il trend
- 🏛️ Il **fornitore di dati** (BCE, FED, BOE, ecc.)
- 📊 Un **mini-grafico**
