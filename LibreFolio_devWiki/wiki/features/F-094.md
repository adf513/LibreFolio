# F-094 — Sync Date Range Dialog

## What it does
A date range selector dialog shown before data sync operations (both FX and Asset). The dialog warns the user if the selected range would overwrite existing data and lets them choose a narrower range for targeted re-sync.

## Implementation notes
- Applies to both FX Sync All modal and Asset sync controls
- Warning banner when selected range overlaps existing data
- Medium priority
