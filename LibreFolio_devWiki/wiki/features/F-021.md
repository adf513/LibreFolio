# F-021 — FX List View dual view

## What it does
The `/fx` page showing all configured currency pairs in two interchangeable views: a card grid (`FxCard.svelte`) and a data table (`FxTable.svelte`). View preference is persisted in localStorage. Each card shows the pair name, current rate, delta %, last sync date, and a mini chart. The page includes "Add Pair", "Sync All", and bulk-delete actions.

## Layer breakdown
**Frontend**: `frontend/src/routes/(app)/fx/+page.svelte`, `frontend/src/lib/components/fx/FxCard.svelte`, `frontend/src/lib/components/fx/FxTable.svelte`, `frontend/src/lib/components/fx/FxPairAddModal.svelte`, `frontend/src/lib/components/fx/FxSyncModal.svelte`

## Dependencies

- [[F-066]] Zodios API Client — typed API calls

## Implementation notes
- Dual view toggle (card/table) with state persisted in localStorage key `fx-view-mode`
- `FxTable` uses the generic `DataTable` component with sortable columns
- Inversion toggle per pair (show EUR/USD or USD/EUR) — stored in `fxCardInversionStore.ts`
- "Sync All" triggers `POST /fx/sync` for all active pairs
