# F-092 — Default Language/Currency for New Users

## What it does
Global settings that define the default language and currency applied to newly registered users. When a user registers, their settings are initialized from the global defaults rather than hardcoded EN/USD.

## Implementation notes
- `get_default_language()` and `get_default_currency()` in `backend/app/services/global_settings_service.py`
- Applied in user registration flow
- Low priority
