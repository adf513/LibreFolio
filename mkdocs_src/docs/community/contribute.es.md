# :heart: Apoya LibreFolio

LibreFolio es un proyecto **de código abierto**, publicado bajo licencia AGPL-3.0. El código fuente está disponible libremente, y cualquier persona con las competencias y la infraestructura necesarias puede instalarlo y utilizarlo de forma autónoma — esa es la belleza del código abierto.

Si usas LibreFolio y lo encuentras valioso, nos encantaría contar con tu apoyo — ya sea a través de **código**, **ideas** o una **pequeña donación**. Cada contribución impulsa el crecimiento del proyecto.

## :computer: Contribuye con Código

Aceptamos contribuciones de cualquier tamaño:

1. **Haz un fork** del [repositorio](https://github.com/Alfystar/LibreFolio)
2. **Crea una rama** para tu funcionalidad o corrección
3. **Envía una Pull Request** con una descripción clara

Consulta el [Manual del Desarrollador](../developer/) para detalles de arquitectura, convenciones y guías de pruebas.

## :globe_with_meridians: Próximamente — LibreFolio Cloud

Para quienes quieran usar LibreFolio pero no tengan el tiempo, las competencias o la infraestructura para autoalojarlo, estamos planificando una **plataforma alojada** — **LibreFolio Cloud**. Ofrecerá todas las mismas potentes funcionalidades sin ninguna configuración técnica, además de futuras **analíticas impulsadas por IA** para ayudarte a tomar decisiones de inversión más inteligentes.

Para sostener la infraestructura, el mantenimiento y el desarrollo continuo, la plataforma cloud se ofrecerá como **servicio de suscripción** — el precio se determinará más adelante.
