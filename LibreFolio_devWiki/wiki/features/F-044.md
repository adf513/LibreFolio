# F-044 — Benchmark Signals Linear/Compound/Sine

## What it does
Three synthetic benchmark signals that generate theoretical reference curves anchored to the first data point of the chart. Useful for comparing actual performance against a linear return target, a compound growth scenario, or for aesthetic/demo purposes (sine wave). All three are in the `benchmark` category.

### Linear (`LinearSignal.ts`)
Straight line at a constant annual growth rate. Formula: `y = y0 × (1 + offset/100 + rate × t)` where `t = days/365`.
- `annualRate` (default 2, suffix `%/yr`), `offset` (default 0, suffix `%`)

### Compound (`CompoundSignal.ts`)
Exponential growth curve. Formula: `y(t) = y0 × (1 + offset/100) × (1 + rate)^t`. Computed iteratively as daily multiplication factor to avoid N `Math.pow()` calls.
- `annualRate` (default 8, suffix `%/yr`), `offset` (default 0, suffix `%`)

### Sine (`SineSignal.ts`)
Sinusoidal oscillation around a baseline. Formula: `y(t) = y0 × (1 + offset/100 + (amplitude/100) × sin(2π × days / period))`.
- `amplitude` (default 15, suffix `%`), `period` (default 45, suffix `days`), `offset` (default 0, suffix `%`)
