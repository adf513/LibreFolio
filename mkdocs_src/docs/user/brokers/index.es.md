# 🏦 Corredores

Un **Corredor** en LibreFolio representa una cuenta de corretaje — el lugar donde tienes tus inversiones (por ejemplo, Interactive Brokers, Degiro, una cuenta bancaria).

Todas las transacciones, informes y datos importados están vinculados a un corredor. Necesitas al menos un corredor para comenzar a seguir tu cartera.

## ➕ Crear un Corredor

1. Navega a la página **Corredores** desde la barra lateral
2. Haz clic en **"Nuevo Corredor"**
3. Completa los detalles: nombre, moneda base y, opcionalmente, un icono

## 🔧 ¿Qué Puedes Hacer con un Corredor?

| Característica | Descripción | Estado |
|---------------|-------------|--------|
| **Ver detalles** | Nombre, moneda, fecha de creación, icono | ✅ Disponible |
| **Subir informes** | Archivos CSV/Excel de tu corredor real | ✅ Disponible |
| **Compartir acceso** | Invitar a otros usuarios con permisos basados en roles | ✅ Disponible |
| **Importar transacciones** | Análisis automático de informes subidos a través de BRIM | 🔜 Próximamente |
| **Ver transacciones** | Listar, filtrar y gestionar operaciones importadas | 🔜 Próximamente |

!!! note "En desarrollo"

    La página de detalles del corredor y la interfaz de gestión de transacciones están en desarrollo activo. Actualmente puedes crear corredores, subir informes y compartir acceso. La importación y visualización completa de transacciones se añadirán en fases próximas.
