# F-041 — Bollinger Bands Signal

## What it does
Bollinger Bands adaptive volatility envelope on the primary Y-axis. Renders three lines: the middle SMA (expected value), and upper/lower bands at ±k standard deviations. When bands squeeze, a breakout is expected. With k=2 and Gaussian assumption, 95.4% of values fall within the bands. Displayed as a shaded confidence band with the SMA line visible.

## Implementation notes
- `signalType`: `bollinger`, `category`: `indicator`, `icon`: 📏
- `yAxisIndex = 0` — primary (same scale as price)
- **Parameters**:
  - `period` (number, default 20, range 2–200, suffix `days`) — SMA window
  - `multiplier` (number, default 2.0, range 0.5–5.0, step 0.1, suffix `σ`) — band width
- Uses `renderMulti()` — 3 series: middle SMA + upper band + lower band (area fill between bands)
- Computed iteratively O(N) using sliding window for SMA and σ
- Docs: `financial-theory/technical-analysis/indicators/bollinger-bands.en.md`
