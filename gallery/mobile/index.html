    📱 Mobile Gallery

    Experience LibreFolio's responsive mobile interface. Screenshots automatically adapt to your selected theme and language.

    Theme & Language

    Use the theme toggle in the header (☀️/🌙) to switch between light and dark mode. Use the language selector (🇬🇧) in the header to view screenshots in different languages.

    Mobile Optimized

    LibreFolio is fully responsive and works great on smartphones and tablets. The interface automatically adapts to smaller screens with a collapsible navigation menu.

    🔐 Authentication

    🔑 Login Page

    Clean and accessible login on mobile devices.

    📝 Registration

    Easy account creation with password strength feedback.

    📊 Dashboard

    Your portfolio at a glance — key stats, allocation charts, and a clean empty state for new users.

    ☰ Navigation Menu

    Full navigation accessible via the hamburger menu.

    📒 Transactions

    📋 Transaction List

    All your recorded financial operations in a filterable, sortable table view.

    ➕ Add / Edit Transaction

    Record any operation — buy, sell, dividend, deposit, transfer, FX conversion — with asset lookup. The form adapts to the selected type.

    🔗 Transaction Picker

    Search and link an existing transaction to create paired operations (e.g. FX trades, transfers).

    ✂️ Split Confirm

    When linking transactions with mismatched quantities, the system proposes a split — previewing the result before confirming.

    🔀 Merge / Promote

    Merge related operations into a single consolidated transaction.

    🗑️ Delete Linked Pair

    When deleting a transaction that is part of a linked pair, you can choose to remove just one leg or both.

    📥 Import

    Import transactions from your broker's export files — quick modal or full wizard.

    ⚙️ Settings

    🎛️ User Preferences

    All settings accessible on mobile with the same functionality.

    🛡️ Global Settings (Admin)

    System-wide configuration on mobile.

    🕐 Market Data Scheduler — Config

    Configure the automatic market data sync daemon on mobile.

    📜 Market Data Scheduler — Log

    View the history of scheduler executions on mobile.

    ℹ️ About

    System information at your fingertips.

    🔐 Password Change

    Secure password changes on the go.

    👤 Profile

    Your profile, avatar, and account settings on mobile.

    📁 Files

    📂 Static Resources

    Manage your uploaded files on mobile.

    🗂️ Static Resources - Grid View

    Visual file browsing with image previews on mobile.

    📋 Broker Reports (BRIM)

    Import and manage broker reports.

    👁️ File Preview

    Preview any static file directly in the browser — CSV grid, image, PDF, markdown, and plain text.

    🖼️ Media & Upload

    ✂️ Image Editor

    Crop, rotate, zoom and flip images on mobile. Supports presets for avatars and icons.

    🎨 Asset Picker

    Choose an existing file, enter a URL, or upload a new image.

    📤 File Uploader

    Drag-and-drop or browse to upload files. Images open the crop editor, other files can be renamed.

    🏦 Brokers

    📋 Broker List

    Your brokerage accounts with touch-friendly cards.

    🔍 Broker Detail

    Detailed broker view optimized for mobile.

    ✏️ Broker Edit

    Edit broker details — name, icon, base currency, and configuration.

    🤝 Broker Sharing

    Manage co-ownership and access roles with the sharing modal, optimized for mobile.

    💼 Assets

    📋 Asset List

    All your tracked financial instruments, optimized for mobile.

    🔎 Asset List Filtered

    Filter assets by name, type, or search text on mobile.

    📈 Detail Chart

    Interactive price chart with date range and currency toggle. Switch between line and candlestick view.

    📡 Signals Overlay

    Technical indicators (EMA, RSI, MACD, Bollinger) on the price chart — each in its own sub-panel.

    📏 Measures Panel

    Click-to-click measurement tool on mobile — view delta, %, and annualized return over any date range.

    🗂️ Classification & Metadata

    Sector pie chart and geographic world map for assets with rich metadata.

    ✏️ Data Editor

    Inline price data editor on mobile.

    ➕ Create Asset

    Create a new asset on mobile — or launch from the Import Wizard with pre-filled data.

    💱 FX Rates

    📋 FX List

    All configured currency pairs with mini-charts, optimized for mobile.

    🔎 FX List Filtered

    Currency pairs filtered by base currency.

    ➕ Add Pair — Direct Routes

    Add a new currency pair with direct provider routes (e.g. USD/CHF via FED).

    🔗 Add Pair — Chain

    Add a pair using conversion chain through an intermediate currency (e.g. NOK/CHF via EUR).

    🔧 Provider Config

    Configure provider priority and conversion routes.

    🔄 Sync All

    Sync all configured pairs from their providers.

    📈 Detail Chart

    Full-size interactive chart for a currency pair.

    📡 Signals Overlay

    Technical indicator signals panel overlaid on the chart.

    📏 Measures Panel

    Click-to-click measurement tool.

    ✏️ Data Editor

    Inline data editor for rate values on mobile.

    📄 CSV Import

    Import rate data from CSV files.

    ⚙️ Chart Settings

    Chart aesthetics and signal overlay settings.
