# ⚖️ Crédits & Mentions légales

## 🏆 Crédits

LibreFolio est un travail de passion, conçu pour offrir aux individus la souveraineté sur leurs données financières.

* **Emanuele Alfano** - *Créateur & Développeur Principal*
 * Concept, Architecture, Backend (FastAPI), Frontend (SvelteKit), et DevOps.
* **Nicoleta Alfano** - *Graphisme & Design*
 * Design du logo, palette de couleurs et identité visuelle.

### 🤖 Développement Assisté par IA

Ce projet a été développé avec le support d'assistants de codage IA avancés, en utilisant une méthodologie structurée **Plan -> Dev -> Test -> Revue**.

* **Méthodologie** : Chaque fonctionnalité commence par un plan écrit détaillé, suivi d'un développement itératif, de tests rigoureux et d'une revue.
* **Transparence** : Le dépôt contient l'historique complet de ces plans dans le dossier `LibreFolio_developer_journal`, servant de témoignage du processus de développement et de ressource pour d'autres développeurs.

### 🌐 Traductions de la Documentation

La documentation multilingue (italien, français, espagnol) a été générée automatiquement à l'aide de [**Aphra**](https://github.com/DavidLMS/aphra) — un flux de travail de traduction basé sur des agents et open-source créé par David LMS. Le pipeline de traduction intègre le flux Analyse → Traduction → Critique → Affinement d'Aphra avec un système de diff structurel personnalisé pour l'assurance qualité.
