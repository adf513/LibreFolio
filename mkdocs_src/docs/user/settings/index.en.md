# ⚙️ Settings

LibreFolio's Settings panel lets you personalise the application and, if you are an administrator, manage global configuration.

Settings are split into three main areas:

- 👤 **[User Preferences](preferences.md)** — Language, currency, theme, profile details, and security password settings.
- 🛡️ **[Global Settings](admin.md)** — Instance-wide configuration, self-registration toggle, session timeouts, and the background market data scheduler.
- ℹ️ **[About](about.md)** — License info, version numbers, and links to official resources.
