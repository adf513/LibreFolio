# F-078 — User Filter in Files Page

## What it does
Admin-only filter on the Files page to show files uploaded by a specific user. Requires a backend endpoint to list users (for the filter dropdown) and uses the pre-wired `get_upload_by_user()` function in the upload service.

## Implementation notes
- Backend: `GET /api/v1/admin/users` endpoint needed (or reuse existing user list)
- Backend: `uploaded_by_user_id` column filter on uploads query
- `get_upload_by_user()` in `backend/app/services/static_uploads.py` already pre-wired
- Frontend: user dropdown filter on Files page (admin-only visible)
- Medium priority
