!!! info "Beta"

    This plugin is in **Beta** — tested with sample files but edge cases may exist.

## How to Export

1. Log in to [Charles Schwab](https://www.schwab.com).
2. Go to **Accounts → History → Export** and download as CSV.

## Notes

- US-formatted CSV (MM/DD/YYYY dates, USD amounts).
- Equity trades and dividends supported.
