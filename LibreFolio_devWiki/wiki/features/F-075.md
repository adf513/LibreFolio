# F-075 — TanStack Table v9 Migration

## What it does
Migrate the DataTable from the current custom Svelte 5 adapter for TanStack Table v8 to the official `@tanstack/svelte-table` adapter when v9 becomes stable. The v8 official adapter is incompatible with Svelte 5 runes, so a custom adapter was built as a workaround.

## Implementation notes
- Current location: `frontend/src/lib/tanstack-table/` (custom Svelte 5 adapter)
- Target: `@tanstack/svelte-table` official package (v9, not yet stable at time of planning)
- Migration is low risk but requires v9 to reach stable release first
- All DataTable components in `lib/components/table/` use the custom adapter
