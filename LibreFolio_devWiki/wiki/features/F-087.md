# F-087 — Smooth Signal Line Style

## What it does
Adds a cubic spline ("smooth") rendering option for signal overlay lines in the chart. Currently all signal lines use linear interpolation between data points. A smooth option would use cubic spline for a more aesthetic curve.

## Implementation notes
- ECharts `series.smooth: true` for built-in smoothing
- Add `smooth` param to `ChartSignal` base class or per-signal style config
- Low priority — aesthetic improvement only
