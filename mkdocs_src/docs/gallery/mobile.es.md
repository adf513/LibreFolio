# 📱 Galería Móvil

Experimenta la interfaz móvil responsiva de LibreFolio. Las capturas de pantalla se adaptan automáticamente al tema y lenguaje seleccionados.

!!! tip "Tema e Idioma"

    Usa el **interruptor de tema** en el encabezado (☀️/🌙) para cambiar entre el modo claro y oscuro.
    Usa el **selector de idioma** (🇬🇧) en el encabezado para ver las capturas de pantalla en diferentes idiomas.

!!! info "Optimizado para Móviles"

    LibreFolio es totalmente responsivo y funciona perfectamente en smartphones y tablets. La interfaz se adapta automáticamente a pantallas más pequeñas con un menú de navegación colapsable.
