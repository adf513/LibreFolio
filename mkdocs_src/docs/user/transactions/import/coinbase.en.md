!!! info "Beta"

    This plugin is in **Beta** — tested with sample files but edge cases may exist.

## How to Export

1. Log in to [Coinbase](https://www.coinbase.com).
2. Go to **Taxes → Reports → Generate Report → Transaction History** and export as CSV.

## Notes

- Crypto transactions only.
- USD and EUR base currencies supported.
