# F-055 — Portfolio Charts

## What it does
ECharts-based portfolio visualization charts on the dashboard: a **Portfolio Growth chart** (invested vs market value over time, dual-series line), an **Asset Dual-Axis chart** (price on left Y, per-transaction gain/loss on right Y with cumulative area + sell arrows as ECharts markPoints), and an **Allocation donut chart** (by asset type).

## Layer breakdown
**Frontend**: Planned — `frontend/src/lib/components/` — `PortfolioChart.svelte`, `AssetDualAxisChart.svelte`, `AllocationChart.svelte`

## Dependencies
- [[F-054]] Dashboard KPI — charts live on the dashboard page
- [[F-046]] Transaction API — data source for gain/loss calculations

## Enables
- (visual portfolio analysis — no downstream)

## Implementation notes
- `PortfolioChart`: 2 ECharts series (total invested vs market NAV), range selector
- `AssetDualAxisChart`: dual Y-axis, sell events as ECharts `markPoints` (↓ arrows), cumulative P&L area fill
- `AllocationChart`: `SemiDonutChart`-style ECharts donut grouped by `asset_type`
- All use same ECharts instance pattern as existing `PriceChartFull`
- Phase 9 — detailed design in `plan-phase05-to-08-upgrade.md §8`
