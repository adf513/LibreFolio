# F-015 — FX Provider Registry ECB/FED/BOE/SNB

## What it does
Registry of four central-bank FX rate providers, each covering a different base currency. Providers are auto-discovered at startup via the `@register_provider(FXProviderRegistry)` decorator. The registry exposes a unified interface for listing providers and fetching rates.

Providers:
- **ECB** (`ecb.py`) — European Central Bank, base EUR, XML feed
- **FED** (`fed.py`) — Federal Reserve, base USD, JSON API
- **BOE** (`boe.py`) — Bank of England, base GBP, JSON API
- **SNB** (`snb.py`) — Swiss National Bank, base CHF, JSON API

## Layer breakdown
**Backend**: `backend/app/services/fx_providers/{ecb,fed,boe,snb}.py`, `backend/app/services/fx.py` (abstract `FXRateProvider` base class), `backend/app/services/provider_registry.py` (`FXProviderRegistry`)

Icons: `backend/app/services/fx_providers/static/`

## Implementation notes
- Abstract base: `FXRateProvider` in `backend/app/services/fx.py`
- Each provider implements: `code`, `name`, `icon`, `default_base_currency`, `fetch_rates(pairs, date_range)`
- Cache: `_fx_fetch_cache` (TTL 5 min) — prevents provider rate-limiting on repeated syncs
- Provider list endpoint: `GET /api/v1/fx/providers` (MANUAL hidden from this list)
