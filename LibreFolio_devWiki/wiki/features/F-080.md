# F-080 — Candlestick Chart / Volume Bars

## What it does
Adds a candlestick (OHLCV) chart view to the asset detail page, alongside the existing line chart. Volume bars shown in a separate grid below the price chart. Toggle in `ChartToolbar` now unlocked for assets (FX remains line-only).

## Implementation notes
- Architecture: 2-grid layout (price grid + volume grid) with synchronized `dataZoom`
- ECharts candlestick format: `[open, close, low, high]` (NOT standard OHLC order!)
- yAxis: [0]=price, [1]=RSI (if overlay), [2]=MACD (if overlay), [3]=volume
- Volume grid auto-hidden when all volume values are null (`actualShowVolume` guard)
- Color: green=bullish (close≥open), red=bearish, volume bars at 50% opacity
- Overlay signals maintain their original `yAxisIndex` — fully compatible
- `LineDataPoint` extended with optional `open?, high?, low?, close?, volume?` fields
- Tooltip: custom format showing O/H/L/C + formatted volume (K/M suffix)
- Completed: 2026-06-01
