# F-036 — Provider Comparison Modal

## What it does
A modal dialog that shows a side-by-side diff between the asset's current stored metadata and the values returned by the provider during a metadata refresh. Each changed field is highlighted. Users can apply the provider's values, keep the current ones, or cherry-pick field by field.

## Layer breakdown
**Frontend**: `frontend/src/lib/components/assets/ProviderComparisonModal.svelte` (323 lines)

## Dependencies
- [[F-029]] Asset Metadata Refresh — modal is shown after a metadata fetch returns differences

## Enables
- (UX utility — no downstream dependencies)

## Implementation notes
- Uses `OldNew[T]` schema from `backend/app/schemas/common.py` for each diffed field
- Fields diffed: `display_name`, `currency`, `asset_type`, `classification_params` (sector, country, etc.)
- "Apply all" button sets all fields to provider values; individual toggles per field also available
- Changes submitted via `PATCH /assets/{id}` on confirm
