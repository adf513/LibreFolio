## Panoramica

La Dashboard è la **pagina principale** dopo il login. Fornisce una vista consolidata di tutto il tuo portafoglio su tutti i broker — patrimonio netto, allocazione, performance recenti e widget a accesso rapido.

I widget pianificati includono:

- **Patrimonio Netto** — Valore totale del portafoglio nella tua valuta di base
- **Ripartizione Allocazione** — Grafici di allocazione per tipo di asset e distribuzione settoriale
- **Panoramica Performance** — Rendimenti su intervalli di tempo selezionabili (1W, 1M, 3M, YTD, 1Y, All)
- **Transazioni Recenti** — Ultime 5–10 transazioni tra tutti i broker
- **Market Movers** — Le tue posizioni ordinate per variazione percentuale giornaliera
- **Esposizione FX** — Riepilogo dell'esposizione valutaria

---

## Controllo di Connettività

LibreFolio controlla automaticamente la **connettività del backend** quando apri la Dashboard. Se il backend non è raggiungibile, viene mostrato un banner con un pulsante di riconnessione — non vengono visualizzati dati obsoleti.
