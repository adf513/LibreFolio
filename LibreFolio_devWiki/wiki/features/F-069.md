# F-069 — MkDocs Multi-Language Documentation

## What it does
LibreFolio's formal user and developer documentation built with MkDocs Material. Supports four languages (EN/IT/FR/ES) via a suffix strategy (`.en.md`, `.it.md`, `.fr.md`, `.es.md`). Sections: user manual, admin manual, developer reference, financial theory (with LaTeX/KaTeX), and UI screenshot gallery. Served from `mkdocs_src/site/` inside the Docker image.

## Layer breakdown
**Infra**: `mkdocs_src/mkdocs.yml` (main config), `mkdocs_src/docs/` (source files), `mkdocs_src/site/` (build output, gitignored), `mkdocs_src/aphra-pipeline/` (LLM translation pipeline scripts)

## Dependencies
- (standalone documentation system)

## Enables
- [[F-070]] Aphra LLM Translation Pipeline — translates EN source to IT/FR/ES

## Implementation notes
- Suffix strategy: all new docs written in EN (`.en.md`), translations generated by Aphra pipeline
- Developer docs (`docs/developer/`) are EN-only by design
- MkDocs Material features: Mermaid diagrams, LaTeX via KaTeX, admonitions, custom gallery JS
- Admonition rule: **always** empty line between `!!!`/`???` directive and body (Prettier-safe)
- Gallery images served dynamically: `{basePath}/gallery/{viewport}/{lang}/{theme}/{category}/{name}.png`
- Commands: `./dev.py mkdocs build|serve|translate|translate-diff`
