# F-022 — FX Detail Page

## What it does
Per-pair detail page at `/fx/[pair]` showing the full rate history chart, provider configuration, manual rate editor, and price summary statistics. Users can sync rates for the specific pair, add manual rates (for MANUAL-sentinel pairs), edit or delete historical rates, and configure provider assignments.

## Layer breakdown
**Frontend**: `frontend/src/routes/(app)/fx/[pair]/+page.svelte`, `frontend/src/lib/components/fx/FxDataEditorSection.svelte`, `frontend/src/lib/components/fx/FxProviderConfig.svelte`, `frontend/src/lib/components/fx/FxPriceSummary.svelte`, `frontend/src/lib/components/fx/FxDataImportModal.svelte`

## Dependencies
- [[F-016]] FX Pair CRUD — pair data and provider assignments
- [[F-017]] FX Rate Sync — per-pair sync button
- [[F-023]] FX Chart & Signal Overlay — full-height chart

## Enables

## Implementation notes
- URL param `[pair]` is the pair slug (e.g., `eur-usd`)
- `FxDataEditorSection` uses the generic `EditBuffer` pattern for inline rate editing
- `FxProviderConfig` allows reassigning/reordering providers with priority drag
- Inversion toggle inherited from list view
