# F-045 — Measure Tool 2-click delta

## What it does
A 2-click measurement overlay that lets users mark two points on any chart and instantly see the delta between them: absolute difference (Δabs), percentage change (Δ%), number of calendar days, and annualized return. Rendered as a dotted orange line with pin (start) and arrow (end) markers. Managed exclusively by `MeasurePanel.svelte` — not in the signal dropdown.

## Layer breakdown
**Frontend**: `frontend/src/lib/charts/signals/MeasureSignal.ts`, `frontend/src/lib/components/charts/MeasurePanel.svelte` (569 lines)

## Implementation notes
- `signalType`: `measure`, `category`: `measure` (not in registry dropdown)
- **Parameters**: `startDate` (string), `endDate` (string) — dates of the two clicked points
- `MeasurementResult` interface: `startDate`, `endDate`, `startValue`, `endValue`, `deltaAbs`, `deltaPct`, `days`, `annualizedPct`
- Default style: color `#f97316` (orange), `lineWidth: 1`, `lineType: 'dotted'`, `markerStart: 'pin'`, `markerEnd: 'arrow'`
- Not registered via `@register_provider` — `MeasurePanel` creates it directly via `new MeasureSignal(...)`
