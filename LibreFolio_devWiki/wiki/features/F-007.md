# F-007 — Theme System dark/light/auto

## What it does
Three-mode theme system: light, dark, and auto (follows OS preference). The selected theme is persisted in localStorage under key `librefolio-theme` and also synced to the user's server-side settings. The system applies the `html.dark` CSS class to toggle between themes. All component colors use Tailwind CSS 4 custom properties defined in `app.css`, so a single class toggle flips the entire UI.

## Layer breakdown
**Frontend**: `frontend/src/lib/stores/themeStore.ts` (writable store + localStorage persistence), `frontend/src/app.css` (CSS variables in `html.dark`), `frontend/src/lib/components/layout/Header.svelte` (toggle button)

## Implementation notes
- Tailwind CSS 4 config via `@theme {}` in `app.css` — no TS config file
- Brand colors: `#1a4031` (libre-green), `#f5f4ef` (libre-beige)
- `auto` mode uses `window.matchMedia('prefers-color-scheme: dark')` listener
- ECharts charts also respond to theme changes via reactive store subscriptions
