# 💱 Tasas de Cambio (Divisas)

LibreFolio incluye un sistema completo de gestión de **divisas (FX)**. Te permite seguir las tasas de cambio entre cualquier par de divisas, con sincronización automática desde fuentes oficiales de bancos centrales.

---

## 📋 La Página de la Lista de Divisas

Navega a **Tasas de Cambio** desde la barra lateral para ver todos tus pares de divisas configurados:

Cada tarjeta muestra:

- 🏷️ El **par de divisas** con emojis de banderas (ej., 🇪🇺 EUR → 🇺🇸 USD)
- 📈 La **tasa más reciente** y la tendencia
- 🏛️ El **proveedor de datos** (BCE, FED, BOE, etc.)
- 📊 Un **gráfico sparkline**

Puedes **filtrar por divisas** para encontrar rápidamente un par específico:
