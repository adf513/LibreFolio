# ⚖️ Créditos y Aspectos Legales

## 🏆 Créditos

LibreFolio es un proyecto hecho con amor, diseñado para empoderar a las personas con la soberanía de sus datos financieros.

### 👥 Equipo Principal

* **Emanuele Alfano** - *Creador y Desarrollador Principal*
 * Concepto, Arquitectura, Backend (FastAPI), Frontend (SvelteKit) y DevOps.
* **Nicoleta Alfano** - *Ilustración y Diseño*
 * Diseño del logotipo, paleta de colores e identidad visual.

### 🤖 Desarrollo Asistido por IA

Este proyecto se desarrolló con el apoyo de avanzados asistentes de programación en IA, utilizando una metodología estructurada **Planificación -> Desarrollo -> Pruebas -> Revisiones**.

* **Metodología**: Cada funcionalidad comienza con un plan escrito detallado, seguido de desarrollo iterativo, pruebas rigurosas y revisión.
* **Transparencia**: El repositorio contiene el historial completo de estos planes en la carpeta `LibreFolio_developer_journal`, sirviendo como testimonio del proceso de desarrollo y
 un recurso para otros desarrolladores.

### 🌐 Traducciones de la Documentación

La documentación multilingüe (italiano, francés, español) se generó automáticamente utilizando [**Aphra**](https://github.com/DavidLMS/aphra) — un flujo de trabajo de traducción basado en agentes y LLM de código abierto creado por David LMS. El pipeline de traducción integra el flujo de trabajo de Aphra de Analizar → Traducir → Criticar → Refinar con un sistema de diff estructural para garantía de calidad.
